# Download Youtube videos and convert them to MP3
## Required
- youtube-dl
- Lame
- Python 2.x

## Command
 - On Server Side
    ytbmp3Server URL
 - On local Side
    ytbmp3Local [OPTIONS] IP

## DISCRIPTION
 1. Run `ytbmp3Server` on server.
 2. Run `ytbmp3Local` on local for downloading mp3 files you downloaded on server.
